Ceci est la simulation d'un système de cryptographie post-quantique

Le système que nous utilisons (Gentry-Sahai-Waters Encryption) est un système homomorphe fondé
sur l'apprentisage avec erreurs. Cette methode constitue l'une des principales pistes de recherche pour
le développement de la cryptographie post-quantique.

Les différentes étapes de ce systèmes sont détaillées dans les fichiers 'keygen.py', 'enc.py' et 'dec.py'.
Toutes les fonctions utilisées sont dans 'util.py'.
Les librairies python utilisées sont:
    -time
    -random
    -math
    -numpy
    -scipy

Pour commencer la simulation, veuillez lancer le script 'test.py'.
Les messages que nous chiffrons sont des nombres.
